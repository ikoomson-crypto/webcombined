Instructions
  1. Download the template file using the button below
  2. Fill in the invoice information in the Excel file
  3. Each row represents a line item
  4. Multiple rows with the same Customer Name and Invoice Date will be grouped into one invoice
  5. Upload the filled Excel file
Important Notes:
  • Customers must already exist in the system
  • Customer names are case-insensitive
  • Date format: YYYY-MM-DD (e.g., 2024-01-15)
  • VAT Rate and Levy Rate are percentages (e.g., 15 for 15%)
  • If Due Date is not provided, it will default to 30 days after Invoice Date
  • Multiple line items with the same customer and date will be combined into one invoice
